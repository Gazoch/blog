Hack 1: Automatic Front-Door Opener

Once you do something for the second time, it's time for that something to be automated. After rising from a reclined position to buzz in guests at our front gate for the thirtieth time, it was time to bring the gate closer to our fingers. We used an Electric Imp to provide a web interface to a simple relay circuit, which we hardwired to the button that opens our gate. We've document the project here so you can build your own.

Introduciton

The project consists of three parts, which we'll describe separately. We'll then describe how to link them all together, and provide some suggestions and resources for extending the usefulness of this project. Here are the materials you'll need.

  1. Gate Buzzer

    In this section we describe how the gate buzzer works, what you'll need to test in order to implement this (somewhat) safely, and how the relay circuit will actually be wired into the system.

  2. Relay Circuit

    This circuit does what your buzzer's button does: completes your building's buzzer circuit by allowing power to flow between two wires. However, this component uses a relay instead of a button. The relay is controlled by a transistor, which is switched by a separately-powered Electric Imp Breakout Board.

  3. Electric Imp (+software)

    The Electric Imp is an impressive device that combines a microcontroller and wifi shield. The cloud API is easy to program for projects such as this, and makes it simple to connect physical things to the internet. We'll show how to set up the Imp and write the software in this section.

  4. Tying It All Together

    Once you've got a handle on the parts that make up this small system, we'll explain how to link them together to make a working circuit that opens the door from the internet.

  5. Suggestions and Resources

    By this point you'll be able to open your front gate by hitting a webpage with your browser, but there are probably more convenient interfaces, such as a mobile app or command line alias. Also, this is your front door we're talking about, so we'll discuss some security vulnerabilities. Also, it might be nice to get this thing off the breadboard and onto something more solid. We'll talk about soldering this to a proto-board and, as a next step, planning a pcb design. Also, what about a housing for the device?
